# react-github-star-manager

## 服务端

服务端仓库 [star-manager-server](https://github.com/Baifann/star-manager-server)

### 如何使用

```shell
git clone git@github.com:Baifann/star-manager-server.git

cd star-manager-server

npm install

cd src

node server.js
```
